A puzzle app designed to visualize English sentence structures.
The latest version features a matrix-style UI for intuitive tense selection.
It also includes a "My Dictionary" function to save custom irregular verbs, 
making it a truly personalized learning tool.

英語の構造を視覚的に理解できるパズルアプリです。
最新版では、「時間 × 形」を直感的に選べるマトリックス型UIを採用。さらに「マイ辞書機能」により、
不規則動詞を自由に追加・保存して自分だけの学習ツールにカスタマイズ可能です。
