This puzzle allows anyone, even those not proficient in English, to easily change tenses and rephrase sentences.
The latest version features a "My Dictionary" function, allowing users to add and save their own irregular verbs.
Customize it to create your very own English learning app.

英語が苦手な人でも、簡単に時制の変換や、肯定・否定・疑問の言いかえができるパズルです。
最新版では「マイ辞書機能」を搭載し、不規則動詞を自分で自由に追加・保存できるようになりました。
自分だけの英語学習アプリとしてカスタマイズ可能です。
